# SAVEUR Booking

Тестовое задание: страница онлайн-бронирования столика.

## Запуск

```bash
npm install
npm run dev
```

Проверки:

```bash
npm run lint
npm run test
npm run build
```

## Что реализовано

- Форма бронирования: имя, телефон, дата, время и количество гостей.
- Валидация на `onBlur` и при submit, ошибки показываются рядом с полями.
- Имитация отправки через `setTimeout` 1.5 секунды с disabled-кнопкой и лоадером.
- Экран подтверждения с деталями бронирования и повторным бронированием.
- Адаптивная вёрстка для мобильного и десктопного экрана.

## Решения

Форма собрана на `react-hook-form`, чтобы не держать вручную отдельное состояние каждого поля и сохранить понятный submit-flow. Правила валидации вынесены в `src/utils/validation.ts`: так их проще тестировать и переиспользовать. Компоненты разделены на `BookingForm` и `ConfirmationScreen`, а страница `src/app/page.tsx` отвечает только за общий статус бронирования. Стили лежат рядом с компонентами в SCSS Modules, чтобы структура проекта оставалась простой и предсказуемой.
